Минималистичный просмотрщик изображений для Windows, разработанный на платформе WPF.

Проект представляет собой легковесную утилиту для быстрого просмотра графических файлов с простым интерфейсом и базовым набором функций.

## Функционал

* Быстрый запуск и низкое потребление системных ресурсов.
* Минималистичный графический интерфейс с темной темой оформления.
* Поддержка навигации по изображениям внутри выбранной директории.
* Функции масштабирования (зум) изображений.

## Стек технологий

* Платформа: .NET / WPF (Windows Presentation Foundation)
* Язык программирования: C#
* Разметка: XAML

## Сборка и запуск

### Требования
* Операционная система Windows
* .NET SDK (укажите версию, например, .NET 8.0)

### Инструкция по сборке
1. Склонируйте репозиторий:
```bash
git clone https://github.com/amadesya/WPFPic
```

2. Выполните сборку и запуск:
```bash
dotnet run

```
